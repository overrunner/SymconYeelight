# SymconYeelight

Plugin für IP-Symcon zum Steuern einer Yeelight Color Bulb.

Die Yeelight wird auch synchronisiert, wenn diese von einer App bedient wird.

<b>Aktuell ist nur Power und Brightness implementiert! </b>
